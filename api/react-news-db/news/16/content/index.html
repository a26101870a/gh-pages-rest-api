選戰倒數15天，國民黨台北市議員徐巧芯緊咬「無良公關公司」臉書粉專的發文。民進黨台北市長候選人陳時中今天再回應，表示自己一點接觸、一點了解什麼都沒有，硬要做這樣的連結大可不必可以休矣。

徐巧芯9日晚間在臉書貼文，表示看到網路流傳一份資料，內容指出建議找狗仔，跟拍市長候選人行程，底下還有蔣萬安、洪孟楷、徐巧芯等人簽名，指控根本是造假抹黑，急奔松山分局報案。陳時中昨天受訪表示，「有聽到這樣的訊息，但是跟我們完全無關」。

陳時中今天上午到新北投市場、金龍市場掃街拜票，媒體詢問，「無良公關粉專爭議，徐巧芯說跟競總焦糖兩邊的交情蠻好的，她又貼出很多截圖，部長知情嗎？」

對此，陳時中表示，這件事情對競總來講，「我一點接觸、一點了解什麼都沒有，硬要做這樣的連結大可不必可以休矣，應該可以停了」。
